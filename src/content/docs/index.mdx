Pocket Network is decentralized infrastructure for accessing any open data, anywhere. Instead of relying on a single provider, Pocket routes your requests across 5,000+ independent nodes serving 60+ blockchains — with no API key required.

## Quick Start

**Need an RPC endpoint?** Use the public portal — no signup, no API key:

```
https://eth.api.pocket.network
```

Swap `eth` for any [supported chain](/developers/supported-chains). Works with ethers.js, web3.py, viem, wagmi, or any JSON-RPC client. See [Integration Examples](/developers/integration-examples) for code.
