<Callout type="info">
  This roadmap reflects PNF's current strategic priorities. Timelines are targets, not commitments — active protocol development means priorities may shift based on ecosystem needs and DAO governance decisions.
</Callout>

## 2026 Priorities

### Protocol & Infrastructure
- **Bridge Infrastructure** — Evaluate cross-chain bridge infrastructure options for the xPOKT multichain bridge
- **x402 Integration** — Native HTTP 402 payment support for AI agent data access (P1, targeting Q2-26)
- **Multi-chain expansion** — New chain deployments via F-Chains partnerships (Arbitrum, Optimism pipeline)

### Growth & Ecosystem
- **Exchange expansion** — Continued CEX listing outreach and native mainnet integration support
- **Foundation Partnerships** — Dedicated RPC partnerships with chain foundations (Radix confirmed, pipeline active)
- **Developer experience** — Docs rebuild, improved onboarding, integration examples
- **BlockchainQuery** — Claude Extensions Directory listing, expanded chain coverage

### Sustainability
- **Cost reduction** — Targeting 40% additional operational cost reduction in 2026
- **Productive treasury** — Expand F-Chains II staking model to maximize DAO treasury yield
- **Partnership revenue** — Scale Foundation Partnership program to offset infrastructure costs

## 2027 Outlook

- x402 flywheel: AI agents paying for data access autonomously via xPOKT
- SDK ecosystem: Additional SDKs beyond PATH as developer community grows
- Cross-chain governance: expanded Cosmos ecosystem participation as protocol matures
- Protocol maturity: Global Mint TLM phaseout as network reaches self-sustaining relay volume
